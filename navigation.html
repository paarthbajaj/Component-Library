Navigation - Desktop(Simple)

Navbar helps user to navigate throughoout the site. Having easy-to-use navigation is important for any web site. It also works as a roadmap which help users to explore.

Example

Brand Logo Link 1 Link 2 Link 3

Usage

  • Semantically your navbar should be under header tag. Nag tag tells that there are navigation links inside it.
  • Add a class with these properties in order to make your header fixed at the top of website: position-fixed, top,left and right as 0, position absolute and z-index to keep it above all the layers.
  • navigation class is providing padding only so it is optional. White class indicates font color white, white-border class indicates a sloid border which is white in color and link class is used for applying text-decoration: none property.
  • You can remove bg-purple class and can add yours to provide a background color.
