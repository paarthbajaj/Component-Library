Usage

  • card class is given properties like width, position:relative, border-radius, etc.
  • card-badge is given position absolute so that badge can be out of normal flow and top-left properties are given for the posotion on card.
  • card-text-overlay class is used so that text can overlay on card image. Postion: absolute is used here.
  • card-shadow class is provided optimum box-shadow.
