Badges

Badges are displayed on app and folder icons on your Home screen to notify you of something new in an app. What the number represents will depend upon the app - it might be the number of missed phone calls, the number of unread emails, the number of Direct Messages on Twitter, and so on.

Usage

  • By using, m-radius class, you can set a component's border radius 5px.
  • notify and notify-count classes have property of position absolute so the badges can be free of normal flow.
  • notify-count-left class is used for badge on the left.
  • notify-count-right class is used for badge on the right.
