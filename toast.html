Toast

The toast component is like an alert box that is only shown for a couple of seconds when something happens (i.e. when the user clicks on a button, submits a form, etc.).

Example

Can't send photo. Retry in 5 seconds.

Usage

  • toast class includes all the crucial properties like width and other thing. You have to add position properties when using it in the project.
  • toast-bg class is providing the background color only.
