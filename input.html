Input

A text box (input box), text field or text entry box is a control element of a graphical user interface, that should enable the user to input text information to be used by a program.

Usage

  • The required attribute is a boolean attribute. When present, it specifies that an input field must be filled out before submitting the form.
  • The pattern attribute specifies a regular expression that the input element's value is checked against on form submission.
