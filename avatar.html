Avatar

The definition of an avatar is something visual used to represent non-visual concepts or ideas, or is an image that is used to represent a person in the virtual world of the Internet and computers. An example of an avatar is an icon you use to represent you on an Internet forum.

Basic example

Usage

  • By using, rounded-circle class, you can make any element's border in a complete circle shape.
  • m-radius class is provided to make any element's border radius to 5px.
  • flex-row-center-center class gives you 3 properties, namely flex:row, align-content:center and justify-content:center
  • black-border class is used to make border of any element to 2px solid black.
